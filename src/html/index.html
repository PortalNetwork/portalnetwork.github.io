Kaizen

Kaizen lets user learn, deploy, run and manage decentralized applications utilized in languages and frameworks such as Solidity, LLL, Node.js, IPFS, swarmDB, Golem, orbitDB, and many more. User can easily deploy and use decentralized application code with Kaizen CLI by utilizing developer tools like Truffle, Ganache, Web3, open zeppline, git, and Docker.
