One crucial thing Portal Network build is the decentralized Domain Name Service (DDNS), and we hereby proudly present WNS (Wanchain Name Service).

What is WNS?

Wanchain Name Service is a distributed, open, and extensible naming system based on Wanchain blockchain. It primarily changes the current Wanchain addresses (20 bytes long) into name such as “facebook.wan”, which reduces human error by generating readable and memorizable wallet addresses. WNS can also host websites for DWebs for future commercial usage. Furthermore, it can be used to provide metadata about names, such as whois information for the users.

．Domain Marketplace
